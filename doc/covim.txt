COVIM DOCS

To start a new CoVim server:
    :CoVim start [port] [name] (or, from the command line: ./server.py [port])

To connect to a running server:
    :CoVim connect [host address / 'localhost'] [port] [name]
    
To disconnect:
    :CoVim disconnect

To quit Vim while CoVim is connected:
    :CoVim quit or :qall!
